# Pre-planning 與 Planning 會議

## 前言

Pre-planning 會議與 Planning 會議都是為了即將到來的 Sprint 制定開發計畫而設，所以直接將兩個會議寫在一起。在 PaGamO Pre-planning 會在 Sprint 的最後一天下午舉行，會議中將確保下 Sprint 準備開發的內容是明確且準備就緒的，而 Planning 則會在 Sprint 的第一天舉行，團隊會在此會議中承諾要完成的開發目標。
