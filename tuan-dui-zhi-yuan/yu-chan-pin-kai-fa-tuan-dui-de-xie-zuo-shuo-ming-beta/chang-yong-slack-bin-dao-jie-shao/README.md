# 常用 Slack 頻道介紹

{% hint style="info" %}
Slack 是 PaGamO 團隊日常用來溝通的主要工具，當你遇到的議題是部門內部夥伴也不清楚時，就可以將議題重新整理好，並傳至相關 slack 頻道中，後續均會有值週生回覆與討論
{% endhint %}



## 使用常用 Slack 頻道前的須知？

1. 即使是內部資料，也盡量避免在訊息中揭露太多使用者個資。尤其是「**密碼**」完全不能露出。
   1. 如果有需要提供密碼，請確認負責後的值週生後再私訊提供，確認 RD 接收後，也務必刪除密碼訊息。
2. 如遇到教材商品/使用者/任務/訂單有問題時，可以提供下列資訊，有效減少與值週生的來回確認：
   1. 課程世界：課程世界名稱 or 代碼
   2. 教材商品：教材代碼、教材名稱等
   3. 使用者：使用者帳號、UID等
   4. 任務：任務名稱、任務 id 等
   5. 訂單：訂單 id、訂閱商品等

## 如何確認在 pagamo-查資料、pagamo-改資料、bugs-report等 3 個頻道中的討論與處理進度？

1. 當討論串的 emoji 出現 ![](<../../../.gitbook/assets/Screenshot 2024-05-20 at 3.14.37 PM.png>) 時，代表對應的值周生已受理並在處理中
2. 當討論串的 emoji 出現![](<../../../.gitbook/assets/Screenshot 2024-05-20 at 3.16.35 PM.png>)時，代表該議題已被值周生處理完畢
3. 如並未看到上述 emoji，或者當天已有 emoji 但隔天依舊沒有任何值周生回覆時，請進入 bugs-rport 頻道，在頻道後方確認當週值周生是誰後，再回到原討論串進一步 @ 他們確認狀況
